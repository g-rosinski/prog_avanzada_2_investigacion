Proyecto con conexion a MONGO y a REDIS

1. El proyecto consiste en conectar dos bases distintas no relacionales, una conexion a MONGO y otra conexion a REDIS.
	Al seleccionar el boton "Conectar a MONGO" listara las base de datos que tengas
	Al seleccionar el boton "Conectar con REDIS" mostrara una variable cargada en memoria

2. Para levantarlo local es necesario tener una cuenta en MONGO ATLAS y descargar MONGO COMPASS.
	Luego dentro del MONGO COMPASS es necesario copiar la STRING CONNECTION y pegarla dentro del codigo, mas especificacimente en la variable de conexion.
	Es una base local gratuita, es decir que no se compartiran datos ni podran acceder otra persona por mas que tengan mi contrase√±a en el codigo ya que es necesario tenerlo activo al momento de usarlo.
	Cada uno debe levantar MONGO COMPASS con su clave correspondiente.
	

3. Requerimientos:
	Visual Studio Code
	Cuenta en MONGO ATLAS
	Aplicacion de MONGO COMPASS
